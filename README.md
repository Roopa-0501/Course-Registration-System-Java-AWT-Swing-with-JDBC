# Course-Registration-System-Java-AWT-Swing-with-JDBC 

A Java desktop application built using **AWT and Swing** that allows users to **sign up, sign in, and register for courses**, with data stored in a **MySQL database using JDBC**.

---

## Features

- User Sign Up and Sign In
- Authentication using database validation
- Course registration form with:
  - Name
  - Phone Number
  - College
  - Education
  - Course Selection
- Multiple course options using dropdown
- GUI built using Java AWT and Swing
- Smooth screen navigation using CardLayout
- Data persistence using MySQL database

---

## Technologies Used

- Programming Language: Java  
- GUI: Java AWT, Java Swing  
- Database: MySQL  
- Database Connectivity: JDBC  
- IDE: VS Code
